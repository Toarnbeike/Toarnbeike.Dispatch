# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.2.1] - 2026-04-27

## Changed
- Moved interfaces around. Public interfaces to Dispatch.Abstraction, internal interfaces in Dispatch

## [0.2.0] - 2026-04-27

## Added
- Support for notification handling
- Some renaming to indicate it concerns requests.

## [0.1.3] - 2026-04-24

### Tooling
- Introduced CI workflows
- Added dependency on Toarnbeike.Logging for tests

## [0.1.1] - 2026-04-23
- Stricter notnull conditions on TKey and TResult

## [0.1.0] - 2026-04-14

### Added
- Initial release of the dispatching framework
- Full support for manual registration of request handlers
